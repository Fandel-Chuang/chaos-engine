ChaosEngine

面向 3D 多人游戏的、客户端/服务器同构的、纯 C 内核 + Lua 脚本的轻量级开源游戏引擎。

C99 C++17 Lua MIT v0.1.0

🎯 设计理念

⚡ 极度简洁

内核代码量最小化，每个模块只做一件事

🚀 高效执行

纯 C 内核，Cache-Friendly 数据布局，无虚函数开销

🔄 同构逻辑

同一份战斗代码，客户端帧同步 + 服务端状态同步

🧱 严格分层

C 内核 → public_api → C++ 编辑器，三层隔离，单向依赖

🌍 全平台

Windows / Linux / macOS / iOS / Android

🤖 AI 友好

代码结构、命名、日志全部面向 AI 可解析设计

🚀 快速开始

# 克隆
git clone git@gitee.com:zhong-fangdao/chaos-engine.git
cd chaos-engine

# 编译 (需要 CMake 3.20+)
mkdir build && cd build
cmake .. -DCHAOS_BUILD_EDITOR=ON
cmake --build .

# 运行无头模式（服务器/测试）
./bin/chaos_headless

# 运行编辑器
./bin/chaos_editor

⚙️ 编译选项

选项默认值说明
CHAOS_BUILD_EDITORON编译 C++ 编辑器
CHAOS_BUILD_TESTSOFF编译单元测试
CHAOS_BUILD_SAMPLESOFF编译示例程序
